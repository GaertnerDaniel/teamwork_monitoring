### I. Présentation du projet de Miguel Cabarellas-Reboredo 
Le but est de comprendre la dynamique des pêches en utilisant les VMS ou les AIS.  
WMS : Vessel Monitoring System. Méthode la plus utilisée. Donne une position toute les heures.  
AIS : Automatic Identification System. A la base pour les avions. Depuis 2014, on l'utilise pour les bateaux de plus de 15 mètre de long. Plus détaillée car elle donne des points toutes les minutes.  
A partir de ces tracés GPS de bateau, essayer de déterminer les moments de pêche. Pour cela, utilisation de la vitesse (vitesse inférieure à 3 noeuds = pêche). Comparaison aux logbooks pour vérifier. Problème, la diminution de la vitesse peut aussi refléter une autre activité.  

### II. Présentation de l'observatoire  
Leur travail est de :
* collecter
* Stocker
* process
* partager
Uniquement les données des bateaux français.  
Plusieurs types de données : 
* des données d'activité : caractéristiques du bateau, position, pêche ...
* des données biologiques : espèce cibles, taille, poids, gender ratio ...
* des observations 
* des données sur les FAD : position, activation, vitesse ...
